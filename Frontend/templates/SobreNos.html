Uma visão completa para um sucesso ágil

O Scrum360 é uma plataforma educacional interativa, idealizada para estudantes e profissionais que desejam dominar a metodologia Scrum.

O que alcançamos:

Temos como missão capacitar indivíduos e organizações, disseminando conhecimento e incentivando a aplicação de metodologias ágeis, para a promoção da inovação e excelência em projetos diversos.

Desejamos nos tornar a principal referência em educação e certificação em metodologias ágeis, transformando equipes, evoluindo e alcançando resultados de alto impacto.

Guiados por valores sólidos como integridade, colaboração, inovação, simplicidade e transparência, buscamos construir relações de confiança e entregar soluções de forma acessivel a todos os públicos.

Perguntas Frequentes

Nós acreditamos que a educação deve ser de fácil acesso e para todos. Portanto a Scrum360 é uma plataforma totalmente gratuita, assim promovendo a acessibilidade e inovação para todos.

Caso seja da área de tecnologia possuimos nosso código open-source, incentivando uso livre, colaborativo e a inovação do nosso projeto.

Flowbite is first conceptualized and designed using the Figma software so everything you see in the library has a design equivalent in our Figma file.

Check out the Figma design system based on the utility classes from Tailwind CSS and components from Flowbite.

The main difference is that the core components from Flowbite are open source under the MIT license, whereas Tailwind UI is a paid product. Another difference is that Flowbite relies on smaller and standalone components, whereas Tailwind UI offers sections of pages.

However, we actually recommend using both Flowbite, Flowbite Pro, and even Tailwind UI as there is no technical reason stopping you from using the best of two worlds.
